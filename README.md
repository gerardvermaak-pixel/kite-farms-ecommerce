# KITE • Premium Ecommerce

**Reliable Cannabis Supply South Africa**

A modern, ultra-fluid ecommerce experience for Kite Farms — showcasing premium cannabis products from 11 elite cultivars.

## Features
- Lightning-fast add-to-cart with animations
- Slide-in cart with live updates
- Multi-step checkout with progress bar
- Product quick-view modals
- Live search + smart filters
- Fully responsive premium dark-luxury design
- Confetti celebration on successful order
